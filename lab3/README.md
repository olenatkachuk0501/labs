1. install dotnet (Download .NET Core SDK)
```
https://www.microsoft.com/net/download
```
2. run app
```
dotnet run
```
3. go to rest API
3.1 get all, GET 
```
http://localhost:5000/api/users 
```
3.1 get by id '1', GET 
```
http://localhost:5000/api/users/1 
```
3.2 create, POST 
```
http://localhost:5000/api/users/
```
3.2 update by id '1', PUT 
```
http://localhost:5000/api/users/1
```
3.2 delete by id '1', DELETE 
```
http://localhost:5000/api/users/1
```
